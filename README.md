# Saúde em Ordem: Transparência Total

Hotsite estático para transparência do mutirão de exames, com zerômetro, documentos, mapa das unidades móveis, explicação de custos, depoimentos e FAQ.

## Como publicar no GitHub Pages

1. Crie um repositório no GitHub.
2. Envie estes arquivos para a branch `main`.
3. Em `Settings > Pages`, selecione `Deploy from a branch`.
4. Escolha `main` e a pasta `/root`.

O site será publicado pelo GitHub Pages em alguns minutos.
